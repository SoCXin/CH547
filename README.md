## [描述](https://github.com/sochub/CH547/wiki) 

[CH547](https://github.com/sochub/CH547) 具有63K flash 和1K 数据空间，拥有3种规格的封装，兼容3.3V/5V电源，最多4路UART，支持USB外设功能。

江苏沁恒增强型8051内核单片机，在USB功能的基础上添加更丰富的功能，包括ADC和触控按键。
